## 技术栈：

​vue+vue-router+vuex+axios+element-ui+echarts

# Build Setup:


```plain
# 将项目克隆到本地
git clone 。。。

# 进入到项目目录
cd 。。

# 安装依赖
npm install

# 在开发环境运行项目，并且在浏览器打开 
npm run dev

# 项目打包
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test

```
